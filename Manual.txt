Bước 1: Điều hướng đến thư mục Project
cd "D:\Python Project\Project 1.2_Rag PDF Chatbot"
hoặc
mở cmd ở đường link file

Bước 2: Tạo một môi trường ảo mới
python -m venv pdf_rag_env

Bước 3: Kích hoạt môi trường ảo
pdf_rag_env\Scripts\activate

Bước 4: Cài đặt PyTorch với hỗ trợ CUDA 11.8
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

Bước 5: Cài đặt bitsandbytes cho CUDA 11.8 (Quan trọng cho lượng tử hóa GPU)
pip install bitsandbytes==0.41.1 --extra-index-url https://huggingface.github.io/bitsandbytes-wheels/cuda118/

Bước 6: Cài đặt các thư viện còn lại
pip install streamlit transformers accelerate langchain-huggingface langchain-community langchain-experimental langchain-chroma langchain pypdf
pip install sentence-transformers

Các lần tiếp theo:

1. Mở Command Prompt/Terminal.
2. Điều hướng đến thư mục project: cd "D:\Python Project\Project 1.2_Rag PDF Chatbot" hoặc mở cmd ở đường link file
3. Kích hoạt môi trường ảo: pdf_rag_env\Scripts\activate
4. chạy lệnh streamlit run ....pystr
4. Đóng Command Prompt/Terminal hoặc chạy lệnh deactivate, môi trường ảo sẽ tự động bị hủy kích hoạt.
